# Compatibility notes for thumbnail-lab

| Variant | Status | Notes |
| --- | --- | --- |
| OpenClaw | full | Full text-to-brief workflow is supported. |
| ZeroClaw | full | Works well for local text processing. |
| PicoClaw | partial | Useful for short scripts, though large drafts may need scoping. |
| NullClaw | unsupported | Unsupported because the documented workflow assumes local text tooling and richer output handling. |
| NanoBot | full | Python-native text shaping is a strong fit. |
| IronClaw | partial | Sandboxing may constrain file access, but static text processing remains feasible. |
