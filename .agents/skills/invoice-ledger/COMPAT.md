# Compatibility notes for invoice-ledger

| Variant | Status | Notes |
| --- | --- | --- |
| OpenClaw | full | Full local reporting works as documented. |
| ZeroClaw | full | Well suited to small CSV-based finance workflows. |
| PicoClaw | partial | Best when scoped to modest invoice exports. |
| NullClaw | partial | Read-only summaries can work, but storage and CSV handling are more limited. |
| NanoBot | full | Python data processing is a strong fit. |
| IronClaw | full | Static local reports work well in sandboxed environments. |
