# Compatibility notes for focus-block

| Variant | Status | Notes |
| --- | --- | --- |
| OpenClaw | full | Ideal for local journaling and DND-aware work sessions. |
| ZeroClaw | full | Journaling works everywhere; DND support depends on the host desktop. |
| PicoClaw | partial | Journal logging works, but DND integrations are usually unavailable. |
| NullClaw | unsupported | Requires local filesystem writes and operator-managed sessions. |
| NanoBot | full | Strong fit for structured journaling and session summaries. |
| IronClaw | partial | Sandboxed runtimes can log sessions but rarely control host notifications. |
