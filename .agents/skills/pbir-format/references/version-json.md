# version.json

PBIR version metadata. Required file.

**Location:** `Report.Report/definition/version.json`

```json
{
  "$schema": "https://developer.microsoft.com/json-schemas/fabric/item/report/definition/versionMetadata/1.0.0/schema.json",
  "version": "2.0.0"
}
```

Both properties required. Format: `major.minor.0` (patch always 0). Do not change unless intentional.
