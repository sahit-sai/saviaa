# Compatibility notes for prompt-fence

| Variant | Status | Notes |
| --- | --- | --- |
| OpenClaw | full | Full prompt-text review is supported. |
| ZeroClaw | full | Text scanning remains effective on constrained hosts. |
| PicoClaw | partial | Best when narrowed to a specific directory instead of a very large tree. |
| NullClaw | partial | Read-only text inspection can work, but full repository scans are limited. |
| NanoBot | full | Python-native regex scanning works well here. |
| IronClaw | full | Static prompt review fits well with sandboxed execution. |
